DoIT OpenLM License Management Automation
Processing the Report Data Using a Jupyter Notebook

Use OpenLM interface to run and export a report as a csv
On the iMAP Util Server navigate to D:\Scripts\OpenLM_LicenseManagementAutomation
Copy the report file into the folder titled ESSENTIAL_Current_Report_File
Use shift + rt click while in the root project folder on Util and choose the "Open Command Window Here" option
Paste the following command at the terminal prompt
jupyter notebook OpenLM_Production_Process.ipynb
When the notebook opens in a browser, navigate to the Variables section and modify the name of the report file
Follow the instructions at the top of the notebook on how to run the process
